# プライバシーポリシー / Privacy Policy

最終更新日: 2026年2月18日

## Sync Clock

### 日本語

Sync Clock（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。

#### 収集する情報
本アプリは、個人情報を一切収集しません。

- ユーザーが設定した壁紙・写真はデバイス内にのみ保存され、外部サーバーへの送信は行いません。
- アナリティクスツールや広告SDKは使用していません。
- アカウント登録は不要です。

#### 写真ライブラリへのアクセス
本アプリは、壁紙として写真を設定する機能のために写真ライブラリへのアクセスを要求する場合があります。選択された写真はデバイス内にのみ保存され、外部に送信されることはありません。

#### データの保存
すべての設定データ（文字盤スタイル、色、壁紙設定など）はデバイス内のローカルストレージにのみ保存されます。

#### お問い合わせ
本ポリシーに関するご質問は、以下のGitHubリポジトリからお問い合わせください。

https://github.com/nori02/SyncClock/issues

### English

Sync Clock ("the App") respects user privacy and is committed to protecting personal information.

#### Information We Collect
The App does not collect any personal information.

- Wallpapers and photos set by the user are stored only on the device and are never transmitted to external servers.
- No analytics tools or advertising SDKs are used.
- No account registration is required.

#### Photo Library Access
The App may request access to your photo library to allow you to set photos as wallpapers. Selected photos are stored only on the device and are never sent externally.

#### Data Storage
All settings data (clock face style, colors, wallpaper settings, etc.) are stored only in local storage on the device.

#### Contact
If you have any questions about this policy, please contact us through the GitHub repository below.

https://github.com/nori02/SyncClock/issues
